AI-Based Phishing URL Detection System

Overview

This project uses Machine Learning to detect phishing websites based on URL and website-related features. A Random Forest Classifier was trained on the PhiUSIIL Phishing URL Dataset to classify websites as legitimate or phishing.

Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Machine Learning

Dataset

- PhiUSIIL Phishing URL Dataset
- Total Records: 235,795
- Total Features: 56

Project Workflow

1. Data Loading and Preprocessing
2. Feature Selection
3. Train-Test Split
4. Random Forest Model Training
5. Model Evaluation
6. Feature Importance Analysis
7. Confusion Matrix Visualization
8. Classification Report Generation
9. Model Saving using Joblib

Results

- Accuracy: 99.99%
- High Precision and Recall
- Successfully classified phishing and legitimate websites

Features

- Phishing Website Detection
- Feature Importance Analysis
- Confusion Matrix Visualization
- Classification Report
- Saved Trained Model (.pkl)
